## Hook (the bet)
"You spent a whole concept on the statistical hedge ratio `h = Cov/Var` — the share count that makes a two-asset book least wiggly. An option trader quotes a *delta*. Are these the same thing, or two different worlds?" They are **one move in two costumes**: the option **delta is exactly that Cov/Var hedge ratio**, re-struck every step as the stock moves. This is the **cumulative interleaved capstone** — modeled on `lesson-states-streaks.json` (multiple unlabeled recall beats spanning the whole arc) + the `lesson-covariance-6` multi-part mastery that forces tool-selection.

## Core promise (one idea)
**The whole options toolkit is one connected set of moves, told apart on sight.** The headline: the option **delta = the `Cov/Var` hedge ratio** you already know; the Greek **signs** are exact (call Δ +, put Δ −, Γ +, vega +, Θ −); and on an unlabeled problem you must *choose* — delta-hedge vs replicate vs q-price — by reading **what the question asks for**, not by re-running the last formula you saw.

## Misconceptions (Specialist — per-option refutation copy)

**`opt6-bet` (`prediction`, byOption) — "Is the option delta the statistical hedge ratio `h=Cov/Var`?"**
- "No — they're unrelated; one is statistics, the other is calculus (∂V/∂S)." → *"Let's test it — they're the same move in two costumes. The delta is the share count that makes a hedged book locally riskless — exactly what `h=Cov/Var` does, only re-struck each step as S moves. (#21)"* (false)
- "Yes, but only by coincidence at one price." → *"Closer — but it's not a coincidence. Delta IS the (instantaneous) `Cov/Var` hedge ratio; the tree's Δ=(C_u−C_d)/(S(u−d)) is that ratio for the next step. (#21, #13)"* (false)
- "Yes — the delta is `h=Cov/Var`, re-struck each step." → *"Exactly — one risk-cancelling ratio, two surfaces. The capstone proves it on #21 (h=2/3) and the canonical tree (Δ=1/2)."* (true)

**`opt6-gate` (`prediction.gate`, label-stripped, byOption refutational) — `gate.correct = delta-hedging`.** Prompt (surface only): *"A desk is long one option and wants the number of shares to short **right now** so the book barely moves on small wiggles in the stock. Which tool gives that number?"* `optionMethods = ['delta-hedging','linearity-indicators','no-arbitrage-replication','binomial-pricing']` (foils EXACTLY `CONFUSABLE[delta-hedging]`); `options`/labels = `['Delta hedging','Linearity / indicators','No-arbitrage replication','Binomial pricing']` (positionally aligned).
- **Delta hedging** → *"Right — the risk-cancelling share count is the delta = the `Cov/Var` hedge ratio, re-struck each step."* (correct)
- **Linearity / indicators** → *"That averages payoffs as a linear combination — an expected value, not a share count. The question asks how many shares to short, not what the option is worth. (delta-hedging)"* (false)
- **No-arbitrage replication** → *"Static replication rebuilds the whole {Δ shares + bond} to PRICE the option; here you only need today's risk-cancelling share count — that's the delta itself. (delta-hedging)"* (false)
- **Binomial pricing** → *"Re-pricing the tree gives the option's VALUE, not its hedge. Choose by what's asked: a risk-cancelling share count = delta-hedging (`Cov/Var`), not re-pricing. (#21)"* (false)

**`opt6-greeksigns` (`answerEntry`, sign-graded via `greekSign`) — per-field refutations:**
- put Δ, likely error `+`: *"A put LOSES value as the stock rises — its delta is **negative**, ∈(−1,0). (`greekSign('delta','put')=−1`)"*
- Γ, likely error `−`: *"A long option has **positive** gamma (and positive vega) — the value curve bends upward; convexity is on your side. (`greekSign('gamma',·)=+1`)"*
- Θ, likely error `+`: *"Long options bleed time value: theta is typically **negative** for a long position. (`greekSign('theta',·)=−1`)"*
- (Magnitudes N(d1), Γ, vega, Θ are display-only on the `greeksSlider` — grade the sign, never the number.)

**`opt6-win` hint ladder** (accept `2/3`, `−2/3`): nudge "h = Cov / Var, one division" → method "6/9, reduce; flip the sign with Cov" → answer "**2/3**; Cov=−6 → **−2/3** (#21)." Most-likely wrong: `9/6`/`3/2` (inverted the ratio) or `6` (reported Cov). **No float accepted** (2/3 is non-terminating).

**`opt6-exotic` hint ladder** (accept `4/5`, `1/2`): nudge "martingale S₀=1, r=0: replicate a $1-at-H payout by buying 1/H shares" → method "price = 1/H" → answer "H=5/4 → **4/5**; H=2 → **1/2**." Most-likely wrong: `5/4`/`2` (reports H) or `1/5`/`1/2`-confusions.

**`transfer-heldout` hint ladder** (accept w_A `6/7`, w_B `1/7`, Var `27/700`): nudge "same `Cov/Var` minimization, now solve for portfolio weights" → method "w_A=(Var_B−Cov)/(Var_A+Var_B−2Cov)=(9/100−3/100)/(1/25+9/100−6/100)=(6/100)/(7/100)" → answer "**6/7** in A, **1/7** in B; min Var = **27/700**." Most-likely wrong: grading **σ=√(27/700)** (the irrational root — **DISPLAY-ONLY, never accept**) or swapping w_A/w_B.

**`opt6-mastery` Part A** (accept `2/3`): the §2.7 costume change — "the static min-variance hedge h=Cov/Var IS today's option delta." Most-likely wrong: `9/6` (inverted). **Part B** (accept `1/2`): the discrimination — "the desk wants the risk-cancelling **share count** (the delta), not the option's price." Most-likely wrong: `10` (re-priced the tree instead of reading the hedge — the gate misconception (d) resurfacing).

**Misconceptions targeted (feed-forward summary):** (a) "the option delta is unrelated to the statistical hedge ratio" → delta **is** `Cov/Var`, the risk-cancelling share count re-struck each step (`opt6-bet`, mastery Part A); (b) "put delta is positive" → put Δ ∈(−1,0), **negative** (`opt6-greeksigns`); (c) "a long option can have negative gamma" → long Γ **> 0** (and vega > 0) (`opt6-greeksigns`, `opt6-explore`); (d) gate: "the right tool is always re-pricing the tree (`binomial-pricing`) / static replication (`no-arbitrage-replication`)" → choose by what the problem **asks**: a risk-cancelling share count = `delta-hedging` (`Cov/Var`), not re-pricing (`opt6-gate`, mastery Part B).
